# ClimateRisk Pro for QGIS

Climate risk assessment and ESG reporting plugin for QGIS.

## Features

- Flood risk assessment
- Heat risk assessment
- Water risk assessment
- Portfolio assessment
- TCFD reporting
- ISSB S2 reporting
- BRSR reporting
- CSRD reporting

## Data Sources

- ThinkHazard
- NASA POWER
- Aqueduct

## Installation

1. Download ZIP
2. Install via QGIS Plugin Manager
3. Open ClimateRisk Pro from toolbar

## Disclaimer

Screening-level climate risk assessment only.
